## High-Level Third-Party Dependencies

- `native-host/`: Node.js, `ws`, and `easymidi`.
- `browser-extension/`: Chrome Manifest V3 extension APIs and browser Web APIs.
- `Wifi_MIDI/`: ESP8266 Arduino core, WiFiManager, AppleMIDI/rtpMIDI support, Arduino MIDI library, and ESP8266 networking libraries.
- `PENELOOPE/`: Teensy/Arduino platform libraries, SPI, Wire, ADC, CircularBuffer, Adafruit GFX, Adafruit SSD1306, and bundled Adafruit font headers.
- `Link_Drum/`: Arduino MIDI library and AVR program-memory support for the Atmega 328 source.
- `MIDI_Bass_Guitar/`: Microchip PIC16F88 assembler include files and toolchain support.
