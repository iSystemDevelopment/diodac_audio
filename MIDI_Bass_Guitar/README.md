# MIDI_Bass_Guitar

PIC16F88 bass guitar to MIDI converter source and firmware artifacts from DIODAC ELECTRONICS.

## Contents

- `B2M.asm` - PIC16F88 assembly source.
- `B2M.HEX` - compiled HEX firmware artifact.
- `B2M.txt` - source text copy/reference file.
